## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOURUSERNAME/4over-api-integration-starter.git
cd 4over-api-integration-starter
npm install
```

Create your `.env` file:

```env
FOUROVER_API_KEY=your_api_key_here
FOUROVER_API_URL=https://api.4over.com
```

Run your application:

```bash
node src/index.js
```
